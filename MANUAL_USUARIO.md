# 📖 Manual de Usuario
## 1. Inicio
Al entrar verás: **"Busca tu libro"**. Presiona `Comenzar a buscar`.

## 2. Chat IA
Escribe naturalmente: `"¿Tenés libros de Python?"`, `"Clean Code está disponible?"`, `"Hola"`.
⚠️ La IA rechaza temas ajenos a la biblioteca.

## 3. Check-in
Menú `Check-in` → Completa nombre/email → Selecciona libro (opcional) → Confirmar.

## 4. Dashboard
Menú `Dashboard` → Ve métricas de uso, consultas recientes y reservas pendientes.

## 5. Admin
Menú `Admin` → Tabla paginada → Busca → Cambia estado disponible/prestado.

## ❓ Preguntas Frecuentes
- ¿Necesito internet? Sí, solo para la IA. El resto funciona offline.
- ¿Es gratis la IA? Sí, Google Gemini Flash tiene capa gratuita generosa.
- ¿Funciona en móvil? Sí, diseño 100% responsive.
